# CravixGIKI-Food_Ordering_System
Cravix is a custom-built mobile food-ordering application designed specifically to serve the Ghulam Ishaq Khan Institute (GIKI) campus community and nearby local vendors in the Topi/Ghazi areas.

# Project Goals and Features

The primary goal is to provide GIKI students and staff with an **easy, fast, and offline-resilient** platform for campus and local area food ordering.

| User Role | Core Features |
| :--- | :--- |
| **Customer** | Browse vendors and menus, place orders, **real-time order tracking**, ratings and reviews, view past orders. |
| **Vendor** | Manage menus (CRUD), toggle open/closed status, accept/decline orders, update order status (e.g., *Preparing*, *Ready*). |
| **Rider (Optional)** | Accept deliveries, update location, mark order as *Delivered*. |
| **Admin** | Onboard and approve new vendors, view simple analytics, manage the system. |
